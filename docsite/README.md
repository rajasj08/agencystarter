# Agency Starter — Documentation site

Developer documentation for the Agency Starter project. Built with Next.js and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001).

## Structure

- `src/pages/` — Doc pages (Introduction, Getting Started, Architecture, Auth & SSO)
- `src/components/docs/` — Layout, sidebar, content wrapper
- `public/docs-assets/` — Images and diagrams for the docs

The existing `docs/` folder in the project root is unchanged; this site is a separate docsite in `docsite/`.
